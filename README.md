# Flutter学習リポジトリ

このリポジトリは、Flutterの学習のために作成されたチュートリアル用プロジェクトです。

## 📱 現在のプロジェクト

### カウンターアプリ (改良版)
- **機能**: カウンターを2ずつ増加
- **追加機能**: リセットボタン
- **UI**: 日本語対応 + 緑色テーマ

## 🚀 実行方法

```bash
# 依存関係のインストール
flutter pub get

# アプリの実行
flutter run -d linux
```
